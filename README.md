> Design Open Graph images in your browser — text, gradients, image backgrounds, and one-click PNG export.

Stop hand-cropping screenshots for your social previews. OG Image Studio is a
small, fast, client-side editor that renders a pixel-perfect 1200×630 card and
lets you export it as a PNG in one click.

## Install

```bash
git clone https://github.com/kea0811/og-image-studio.git
cd og-image-studio
pnpm install
pnpm dev
```

Then open [http://localhost:3000](http://localhost:3000).

More docs (usage, the rendering API, examples) land with the next commit.
